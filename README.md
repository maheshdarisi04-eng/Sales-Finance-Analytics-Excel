# Sales-Finance-Analytics-Excel
# Procedure
1. Performed data transformation/cleaning in power query and loded the datasets into data model.
2. Established relations between tables in the data model.
3. Created a new dimension table for mapping dates to fiscal year in power query.
4. Created a customer sales report for 3 fiscal years.
5. Created a Market performance report showing customer wise % of target sales achieved.
6. Applied conditional formatting to highlight KPIs sales revenue, target achievement, and market-wise trends.
